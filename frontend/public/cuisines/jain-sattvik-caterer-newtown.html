Tier differences

What each discipline allows.

Confusing? Tell us the event and we'll match the tier. Partners then build menu within that exact discipline.

🪷
Jain
Strictest. No onion, no garlic, no ginger. No root vegetables (no potato, carrot, beetroot, radish). Separate prep, separate utensils.
🕉️
Sattvik
Ayurvedic principles. No onion-garlic. Gentle spices (cumin, turmeric, cardamom). Avoids fermented food, stale leftovers. Yoga / spiritual events.
🌿
Niramish
Bengali ceremony-compliant. No onion-garlic. Allows root vegetables (potato, carrot). Standard for Saraswati Puja, Lakshmi Puja, Annaprasan, Griha Pravesh.

Jain · Sattvik · Niramish FAQs

Questions pure-veg hosts ask.

What's the difference between Jain, Sattvik and Niramish?+

Jain — no onion, no garlic, no root veg (no potato, carrot). Sattvik — Ayurvedic, no onion-garlic, gentle spices. Niramish — Bengali ceremony, no onion-garlic, allows root veg like potato.

How much do these menus cost?+

Niramish ₹350-₹450/plate. Sattvik ₹400-₹500/plate. Strict Jain ₹450-₹600/plate.

Common events?+

Saraswati Puja, Lakshmi Puja, Annaprasan ceremony day, Griha Pravesh, ekadashi, Brahmin family events, Jain community events.

Strict Jain experienced partners?+

Yes — separate prep, separate utensils, no root veg. Confirm strict-Jain on WhatsApp; we filter to verified specialists.

Typical menu?+

Jeera rice, dal (panchratan/moong), sabzi (mixed veg, paneer), puri/luchi, raita, papad, dessert. Jain uses raw banana/arbi instead of potato.

Minimum batch?+

25 plates. Ceremonies typically 30-100; Jain weddings 200-500.
