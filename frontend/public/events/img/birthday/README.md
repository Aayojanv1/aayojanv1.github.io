the on-page placeholders automatically — no code change needed.

Guidelines for best results:
- Landscape, ~4:3 ratio, min 800×600px (the grid crops to 4:3).
- Real plating from your founding kitchens beats stock — it's the authentic signal.
- Keep each file under ~300KB (compress at e.g. squoosh.app) so mobile stays fast.
- .jpg is referenced; if you use .webp, update the src in the page (or ask me to).

Until a file is added, that card shows an elegant gradient + emoji + caption, so the
page always looks intentional.
